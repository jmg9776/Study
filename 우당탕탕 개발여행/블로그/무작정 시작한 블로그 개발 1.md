### 블로그 구축: GitHub Pages로 전환

원래 목표는 블로그를 만드는 것이었습니다. 하지만 AWS와 Spring Boot를 사용한 백엔드 구축은 비용과 이미지 업로드 같은 문제로 어려움을 겪었죠. 그러던 중 GitHub Pages를 사용해 블로그를 운영하는 사람들을 발견했습니다.
